Project Overview

Goal: Develop a decentralized guild-based platform where users (mages) can:

  • Join guilds, take on tasks, and earn rewards (cryptocurrency or tokens) via Solana.
  • Integrate Solana blockchain for fast, low-cost payments, creating a task marketplace where jobs are posted and completed.
  • Implement a token economy (Solana-based tokens) for rewarding users and managing transactions.

Key Features

  • User Management (Mage Class): Registration, login, profile management, and roles (mage, guild leader, admin).
  • Guild System: Users can join and create guilds, with guild leaders posting tasks and jobs for mages to complete.
  • Task Marketplace: A marketplace where guilds post tasks/jobs and mages can apply for them.
  • Solana Token Economy: Earn rewards in Solana-based tokens for completing tasks.
  • Transaction System: Users will be able to view their earnings, track their transaction history, and make withdrawals.
  • Admin Dashboard: Admins can manage users, guilds, tasks, and oversee the flow of tokens.
  • Notifications: Real-time notifications for job assignments, task updates, and guild events.
  • Security: User authentication, blockchain-based wallets, and secure transaction handling.
